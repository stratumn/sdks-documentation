<TabItem value="js">

- In Node.js

The preferred way to install the Stratumn SDK for Node.js is to use the
[yarn](http://yarnpkg.com) package manager for Node.js. Simply type the following
into a terminal window:

```sh
yarn add @stratumn/sdk
```

</TabItem>
<TabItem value="csharp">

- Using NuGet console manager

The preferred way to install the Stratumn SDK is to use the
[NuGet](https://www.nuget.org) package manager. Simply type the following
into a NuGet console window :

```sh
Install-Package Stratumn.SDK
```

</TabItem>
