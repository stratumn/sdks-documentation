游니 Installing

  • JavaScript
  • C#
  • Java
  • In Node.js

The preferred way to install the Stratumn SDK for Node.js is to use the yarn package manager for Node.js. Simply type the following into a terminal window:

yarn add @stratumn/sdk
  • Using NuGet console manager

The preferred way to install the Stratumn SDK is to use the NuGet package manager. Simply type the following into a NuGet console window :

Install-Package Stratumn.SDK

Required version : Java >= 8

  • Maven
<dependency>
<groupId>com.stratumn</groupId>
<artifactId>sdk</artifactId>
<version>0.2.1</version>
</dependency>
